## рҹҺҜ What It Does

Grid07 Cognitive Engine is an intelligent AI backend that routes user queries through semantic vector matching, orchestrates multi-step reasoning using LangGraph state machines, and retrieves context-aware responses through a RAG (Retrieval-Augmented Generation) pipeline вҖ” all secured with multi-layered prompt injection defense.

## вңЁ Key Features

- **Vector Routing** вҖ” Semantic similarity-based query routing to specialized agent nodes
- **LangGraph Orchestration** вҖ” Stateful multi-agent workflows with conditional branching
- **RAG Pipeline** вҖ” Context retrieval from vector databases for grounded responses
- **Prompt Injection Defense** вҖ” Multi-layered security against adversarial prompt attacks
- **Modular Design** вҖ” Easily extensible agent nodes and routing logic
